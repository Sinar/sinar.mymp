sinar.mymp Installation
-----------------------

To install sinar.mymp using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``sinar.mymp`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        sinar.mymp

* Re-run buildout, e.g. with:

    $ ./bin/buildout
